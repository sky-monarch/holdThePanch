# Долгосрочное домашнее задание (игра "Hold The Panch")

Общее описание

Игра: 2D платформер, одиночный режим.

Жанр: Action / Platformer.

Платформа: Windows Desktop (экспорт .exe).

Движок: Godot 4.5.1.

Цель игрока: пройти 5 уровней, победив всех врагов и финального босса.

## Этап I
Время на этап: **150 часов**

### Цель

Необходимо разработать игру (2д платформер) для одного пользователя против компьютера. Правила игры можно найти ниже в ТЗ.

### Задание

На этом этапе необходимо реализовать desktop-приложение c графической отрисовкой процесса игры и реализацией основной логики игры.
#### Функционал, который должен быть реализован:
1. Игровое меню
	* кнопка **В БОЙ!** - запускает сцену выбора игрового режима(одиночный или многопользовательский). **ВАЖНО!** игрок может подгрузить данные из последнего сохранения для запуска многопользовательской версии;
	* кнопка **НАСТРОЙКИ** - изменение громкости музыки, выбор уровни сложности(хард, тяжело, нормально, легко), изменение размера экрана на полное или оконное;
	* кнопка **О СОЗДАТЕЛЯХ** - плавное появление информации о разработчиках игры, которая состоит из фамилии, никнейма в github и персожей игры(отссылка игру - минибоссы, а сам босс - секрет, узнаешь в конце);
	* кнопка **ВЫХОД** - завершает работу приложения.
2. Игровой процесс
* визуальная часть:
	* отображение счётчика количества жизней в виде полоски здоровья над игроком
	* отображение текущего уровня в верхней левой части экрана
	* возможность поставить игру на паузу с отображением соответствующего уведомления
	* возможность продолжить игру после паузы с пропажей уведомления о паузе
	* возможность в игровом процессе венуться в главное меню игры
	* возможность запустить игру заново, сделать рестарт уровня
	* по завершении игры последовательное выполнение следующих событий: 
	* переход на следующий уровень в случае победы на всех уровнях, кроме последнего(предусмотреть не менее 5-ти уровней)
	* в случае победы открывается меню "Победа",в которой игрок может посмотреть информацию о создателях или выйти в главное меню.
* логика игры:
  * Игрок начинает проходить игру с 1 уровня, главная задача Игрока убить всех врагов на уровне, чтобы перейти на следующий
  * Переход осуществляется с помощью портала
  * На 5 уровне Игрок должен убить Босса чтобы пройти игру
  * Сложность игры увеличивается с каждым уровнем, увеличивается количесво врагов, их скорость, урон и количество здоровья
  * Каждый уровень появляются новые враги, количество их жизней также отображается сверху над ними
  * Когда у Игрока заканчиваются жизни, он проигрывает и может либо перейти в основное меню игры, либо перепройти уровень
  * В случае уничтожения Босса Игрок оповещается о победе и после переходит в основное меню
  * в случае выхода игрока в главное меню, не дойдя до конца, игровой процесс не сохраняется, то есть в следующий раз игрок начнет с 1 уровня
3. Конфигурация игры
	* в настройках должна быть возможность изменить громкость музыки и выбрать полноэкранный или оконный режимы

Для каждой описанного действия и бездействия, а также для получения урона и смерти должна быть своя анимация. Для соблюдения баланса Враг не может постоянно бить, дожно быть некоторое время после удара, когда враг не может ударять, также враг не может постоянно получать урон, должно быть определенное время после получения удара, когда враг не может снова получить урон, также
